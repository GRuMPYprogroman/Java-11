Напишите программу, которая принимает на вход число n — количество чисел, 
затем последовательность из n чисел, и находит третье по величине уникальное 
число среди этих чисел.

Описание:

Сначала вводится число n, указывающее количество чисел.

Далее вводится последовательность из n целых чисел.

Программа должна вывести третье по величине уникальное число.

Если третьего уникального числа не существует 
(например, недостаточно уникальных элементов), программа должна вывести: 
"Недостаточно уникальных элементов".
